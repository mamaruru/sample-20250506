
サンプルプログラムを書いて。

postgresql-17
Java 21
Spring Boot 3.4

DBアクセスは、Facadeパターン。
データベースのテーブル名、カラム名はスネークケース。idは自動採番。
Entityクラスのクラス名、項目名はキャメルケース。
Javaの代表的なテンプレートエンジンTymeleafを使う。
一覧画面、登録画面、詳細画面がある。
Mavenのpom.xmlもください。
physical-strategy: org.springframework.boot.orm.jpa.hibernate.SpringPhysicalNamingStrategy　です。
IntelliJ IDEAでこのプロジェクトを起動して確認したい。pom.xmlを含んだソースコードをzipにしてください。
